Finding Groups.

The colors of each 'keyword' help you find a specific group

Blue = category

If a word is in blue, or contains a blue background, it signifies a category. One group can have multiple categories, called sub-categories

Favorite pro football team:

Path: Football > NFL > Cleveland Browns

Red = location

Location. Location. Location - the Hard Rock Cafe in Ibiza is a lot different than the Hard Rock Cafe in D.C. Use a location tag to make sure you have the right one.

Path: Hard Rock Cafe > Ibiza, Spain

Yellow = date

Going to sporting events can be great! They're even better when you have the right date. When planning with others to attending a game, make sure you're all on the same page by adding a date tag.

Path: Washington Nationals > 10/28/2019
